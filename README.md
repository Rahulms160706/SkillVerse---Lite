A collection of mini projects built while learning **JavaScript, React, and basic backend development**.  
These projects focus on practicing core frontend concepts such as state management, UI design, and interactive logic.

---

## 📂 Projects Included

### JavaScript Projects
- **Reaction Time Tester**
  - A simple game that measures how fast a user reacts to a visual cue.

- **Memory Flip Game**
  - A card matching game where players flip cards to find matching pairs.

- **Random Story Generator**
  - Generates random stories using predefined characters, actions, and locations.

---

### React Projects

- **XP Dashboard**
  - A dashboard-style interface that tracks experience points (XP) and progress.

- **Quiz Game**
  - A quiz application built with React that displays questions and tracks the user's score.

- **Pet Simulator**
  - A virtual pet simulation where users manage pet stats like:
    - Hunger
    - Happiness
    - Energy
    - Health  
  - Includes stat decay logic, actions (feed, play, sleep), and a leveling system.

---

### Backend

- **Pet Simulator Backend**
  - Basic backend support for the pet simulator using **Node.js and Express**.
